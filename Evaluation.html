Evaluation Of Project Brief
    BR 1 -
  • I successfully selected a suitable dataset based on passenger values in Irish Airports from the data.gov.ie website.
  • I dropped and renamed data using Pandas, and stored the cleaned data into a suitable format.
    BR 2 -
  • I performed the required analytics, calculated statistics like the min, max, mean, median, and range and created at least two basic visualisations ( bar graph, line graph, etc. ) using Plotly.
  • My visualisations include clear labels and legends, making them easy to understand.
  • I made a Python dictionary ( airportDictionary ) to manage and organise the data.
    BR 3 -
  • I developed a web-based interface using HTML, CSS, and JavaScript, providing a user-friendly way to view and interact with the data.
  • I successfully embedded the visualisations into my web interface, allowing users to view the data.

    AR 1 -
  • I implemented interactive features ( Hover for details, legend highlighting, etc. ) using Plotly, enhancing the user experience on the web interface.
    AR 2 -
  • I created a form with at least 3 different appropriate data types to collect additional information from users.
  • I implemented a data validation to ensure data quality and to store data securely in a database.
  • I summarised the data by making them easier to read and displayed the previous entries to the user.
    AR 3 -
  • I developed sections on the web interface that provides data-driven recommendations related to Irish Airports.


More Time / Skill To Improve Artefact
  • I would look into making my web interface more user friendly, especially those with disabilities. Things like a button to change font size for people with poor sight and a button to change the colour contrast of the web for people with colour blindness.

  • I would implement simulations which will be able to predict outcomes on Irish Airports based on certain situations. Using the information I made about COVID-19, predicted outcomes can be made for future pandemics.
